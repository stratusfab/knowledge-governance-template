# Ingestion Workflow

1. Place new materials into /assets-raw.  
2. Create a temporary Markdown extraction file.  
3. Extract claims, facts, or observations.  
4. Compare extracted claims with canonical files.  
5. Decide what is accurate and current.  
6. Update canonical files.  
7. Move raw assets to /assets-processed or a subfolder.  
8. Commit with a clear explanation.
